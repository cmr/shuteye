Usage

Include the module, create a timespec, call clock_nanosleep

Example

Sleep for 1 uS

use shuteye::*;

let ts = Timespec::from_nano(1000).unwrap();
assert_eq!(ts.get_sec(), 0);
assert_eq!(ts.get_nsec(), 1000);
shuteye::sleep(ts);
